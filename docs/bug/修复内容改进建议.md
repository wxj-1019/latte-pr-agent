### 1.1 路径安全验证
**文件**: `src/projects/router.py:96`
**问题**: `os.path.dirname(project.local_path)` 可能受到路径遍历攻击
**建议**: 添加路径安全验证

```python
# 添加路径安全验证函数
def is_safe_path(base_path: str, target_path: str) -> bool:
    """检查目标路径是否在基础路径内"""
    base = os.path.abspath(base_path)
    target = os.path.abspath(os.path.join(base_path, target_path))
    return target.startswith(base)

# 在创建目录前验证

    raise HTTPException(status_code=400, detail="Invalid local path")
```

```python
def safe_filename(s: str) -> str:
    """将字符串转换为安全的文件名"""
    # 替换不安全字符
    s = re.sub(r'[^\w\-.]', '_', s)
```
